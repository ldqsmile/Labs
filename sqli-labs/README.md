# Usage
运行Lab环境
```
docker-compose up -d
```
运行后，用户访问 http://your-ip:10080/sqli-labs/ 即可看到初始页面,其中your-ip为搭建环境机器的地址，如本地搭建your-ip即为127.0.0.1。

点击 “Setup/reset Database for labs” 进行数据库初始化，返回再点击各个关卡即可开始解题。
